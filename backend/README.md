# Backend

To install dependencies:

```bash
bun install
cp template.env .env
```

To run:

```bash
bun run dev:up
```

To view logs:

```bash
bun run dev:tail
```

To shut down:

```bash
bun run dev:down
```

To view the API documentation, go to [http://frumbers.localhost:3000/swagger](http://frumbers.localhost:3000/swagger)

To interact with the database, go to [http://db.frumbers.localhost:3000/](http://db.frumbers.localhost:3000/)
